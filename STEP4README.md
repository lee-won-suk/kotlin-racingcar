# 자동차 경주

#기능 요구사항

1.
    - 자동차 이름 부여 가능
      - [x] 자동차 이름 입력받는 기능 추가
    - [x] 자동차 이름 5자 초과 불가능
    - [x] 자동차 출력시 이름 같이 출력
    - 쉼표를 기준으로 자동차 이름 구분, 자동차개수도 같이 구함.
      - [x] ,로 구분하고, 구분한 개수만큼 자동차 개수로 값 저장
    - 게임 완료 후 우승자 출력.
      - [x] 제일 거리가 긴 Car 객체를 찾아서 저장후 반환한다.
      - [ ] 우승자를 출력하는 메서드를 ResultView에 추가


#구조
우승자 클래스를 만든다.
Cars 내부에 필드로 두고 리스트를 줄테니 우승자 컬렉션을 반환하도록 만든다.
